# Реализация функциональности видеокружков (Video Messages)

## Обзор
В приложение Komet добавлена возможность отправлять видеокружки (video messages) в чаты. Видеокружки - это короткие видеозаписи, воспроизводящиеся в виде круглых плееров в сообщениях.

## Файлы, которые были изменены

### 1. `lib/api/api_service_chats.dart`
**Добавлена функция:** `sendVideoMessage()`

Эта функция обрабатывает отправку видеокружков на сервер:
- Запрашивает URL загрузки через opcode 82 с типом 1 (видео)
- Загружает видеофайл на сервер через multipart request
- Получает токен видео и ID
- Отправляет сообщение на сервер через opcode 64
- Обрабатывает ошибку `attachment.not.ready` с повторными попытками

**Параметры:**
- `chatId`: ID чата
- `localPath`: путь к видеофайлу
- `durationSeconds`: длительность видео в секундах
- `fileSize`: размер файла в байтах
- `width`: ширина видео (по умолчанию 480)
- `height`: высота видео (по умолчанию 480)
- `thumbnail`: опциональный thumbnail (preview)
- `senderId`: ID отправителя
- `onProgress`: коллбэк прогресса загрузки

### 4. `lib/screens/chat_screen_voice.dart`
**Добавлены методы для работы с видеокружками:**

1. **`_startVideoRecordingUi()`** - начинает запись видео:
   - Проверяет разрешения на запись
   - Создает временный файл
   - Запускает таймер записи
   - Обновляет UI

2. **`_cancelVideoRecordingUi()`** - отменяет запись видео:
   - Останавливает запись
   - Удаляет временный файл
   - Сбрасывает UI

3. **`_toggleVideoRecordingPause()`** - паузирует/возобновляет запись видео

4. **`_sendVideoMessage()`** - отправляет записанное видео:
   - Проверяет файл
   - Показывает прогресс загрузки
   - Вызывает `ApiService.instance.sendVideoMessage()`
   - Удаляет временный файл после успешной отправки
   - Кэширует файл при ошибке для возможности повтора

5. **`_buildVideoRecordingBar()`** - строит UI панель записи видео

6. **`_formatVideoDuration()`** - форматирует длительность видеозаписи

## Поток выполнения

### Запуск записи
1. Пользователь нажимает кнопку камеры в toolbr
2. Вызывается `_startVideoRecordingUi()`
3. Создается новый временный файл .mp4
4. Запускается таймер
5. UI переходит в режим записи

### Во время записи
- Пользователь видит таймер записи
- Может нажать паузу для приостановки
- Может свайпить влево для отмены
- Может нажать кнопку отправки для завершения

### Отправка
1. Пользователь нажимает кнопку отправки
2. Вызывается `_sendVideoMessage()`
3. Файл подготавливается
4. Показывается прогресс загрузки
5. Вызывается `ApiService.instance.sendVideoMessage()`:
   - Запрашивается URL загрузки
   - Загружается видео на сервер
   - Получается токен
   - Отправляется сообщение через opcode 64
6. После успешной отправки файл удаляется
7. UI возвращается в нормальное состояние

### Отображение видеокружка
- Когда сообщение содержит вложение с `_type: 'VIDEO'` и `videoType: 1`
- Система автоматически использует класс `_VideoCirclePlayer`
- Видео отображается как круглый плеер размером 200x200
- При клике видео включается/выключается
- Поддерживается автоматическое воспроизведение без звука в фоне

## Протокол взаимодействия

### Опcode 82 - Получение URL загрузки для видео
**Запрос:**
```json
{
  "type": 1,  // 1 = видео, 2 = аудио
  "count": 1
}
```

**Ответ:**
```json
{
  "info": [
    {
      "url": "https://...",
      "id": 12345,
      "videoId": 12345,
      "token": "..."
    }
  ]
}
```

### Opcode 64 - Отправка сообщения с видеокружком
**Запрос:**
```json
{
  "chatId": 0,
  "message": {
    "isLive": false,
    "detectShare": false,
    "elements": [],
    "cid": 110015741068990,
    "attaches": [
      {
        "videoType": 1,
        "_type": "VIDEO",
        "token": "f9LHodD0c...",
        "width": 480,
        "height": 480,
        "duration": 1262,
        "size": 212501,
        "videoId": 12172191163992
      }
    ]
  }
}
```

## Особенности реализации

1. **Состояния видеозаписи** - отслеживаются отдельно от голоса
2. **UI/UX** - аналогичны голосовым сообщениям для консистентности
3. **Ошибки загрузки** - файлы кэшируются для возможности повтора
4. **Прогресс** - показывается прогресс загрузки на сервер
5. **Отмена** - свайп влево отменяет запись
6. **Пауза** - можно приостановить запись и продолжить

## Интеграция с существующей системой

- Видеокружки используют существующую систему отображения в `chat_message_bubble.dart`
- Класс `_VideoCirclePlayer` для воспроизведения
- Функция `getVideoUrl()` из `api_service_media.dart` для получения URL видео
- Совместимы с кешированием и оптимизацией

## Требования

- `record` пакет для записи видео (уже в зависимостях)
- Разрешения на запись видео и аудио
- Доступ к камере устройства
