# Гайд пользователя: Отправка видеокружков

## Что такое видеокружки?
Видеокружки (Video Messages) - это короткие видеозаписи, которые воспроизводятся в виде круглых плееров прямо в чате. Идеальны для быстрого обмена видеосообщениями, лицом или скринкастами.

## Как записать видеокружок?

### Шаг 1: Откройте чат
Перейдите в нужный чат, в котором хотите отправить видеокружок.

### Шаг 2: Нажмите кнопку видеокамеры
В строке ввода сообщения вы найдете кнопку видеокамеры (значок с видеокамерой). Нажмите на неё.

### Шаг 3: Начните запись
После нажатия кнопки:
- Разрешите приложению доступ к камере и микрофону (если потребуется)
- Появится интерфейс записи с таймером
- Начните записывать видеокружок (рекомендуемая длительность - до 30 секунд)

### Шаг 4: Управление записью
Во время записи у вас есть несколько опций:

- **Пауза** 🔲 - Нажмите кнопку паузы (средняя кнопка справа), чтобы приостановить запись
- **Возобновить** ▶️ - После паузы нажмите ту же кнопку, чтобы продолжить
- **Отмена** ❌ - Свайпните влево по области с таймером или нажмите кнопку удаления для отмены записи
- **Отправить** ➡️ - Нажмите кнопку отправки (стрелка справа), чтобы отправить видеокружок

### Шаг 5: Загрузка на сервер
После нажатия отправки:
- Видеозапись будет загружена на сервер (покажется прогресс загрузки)
- Дождитесь завершения загрузки
- Видеокружок появится в чате как круглый плеер

## Просмотр видеокружков

### Автоматическое воспроизведение
- Видеокружки начинают воспроизводиться автоматически без звука при загрузке сообщения
- Видео циклируется (повторяется)

### Ручное управление
1. **Включить звук** - Нажмите на видеокружок один раз
2. **Выключить звук** - Нажмите ещё раз
3. **Пауза** - Нажимайте на видео, чтобы управлять воспроизведением

## Советы и рекомендации

✅ **Делайте:**
- Записывайте видеокружки длительностью 5-30 секунд для лучшей производительности
- Используйте для быстрого обмена видеосообщениями
- Проверяйте освещение перед записью
- Записывайте в хорошем соединении Интернета для быстрой загрузки

❌ **Не делайте:**
- Не записывайте видео длиннее 2 минут - это может привести к проблемам с загрузкой
- Не отправляйте огромные видеофайлы, если у вас слабое соединение
- Не закрывайте приложение во время загрузки видеокружка

## Ошибки и решение проблем

### "Не удалось начать запись"
- Проверьте, что приложение имеет разрешение на доступ к камере
- Перезагрузите приложение
- Убедитесь, что камера не используется другим приложением

### "Видеофайл не создан или пуст"
- Это редкая ошибка Windows - перезагрузите приложение и попробуйте снова
- Убедитесь, что на диске есть свободное место

### "Не удалось отправить видеокружок"
- Проверьте соединение Интернета
- Нажмите кнопку "Повторить" для повторной попытки отправки
- Если проблема persists, перезагрузите приложение

## Требуемые разрешения

Для работы видеокружков приложение нужны следующие разрешения:

| Платформа | Разрешения |
|-----------|-----------|
| Android | Камера, Микрофон, Память |
| iOS | Камера, Микрофон |
| Windows | Доступ к камере* |
| macOS | Доступ к камере* |
| Linux | Доступ к камере* |

*На десктопных платформах запуск видеозаписи может потребовать дополнительную настройку системы

## Часто задаваемые вопросы

**В: Можно ли выбрать видео из галереи вместо записи?**
А: Нет, текущая версия поддерживает только запись видеокружков через камеру.

**В: Какой максимальный размер видеокружка?**
А: Размер ограничен только максимальным размером файла на сервере. Рекомендуется не превышать 50 МБ.

**В: Сохраняются ли видеокружки локально?**
А: Нет, они хранятся только на сервере. Если вы удалите сообщение, видеокружок тоже будет удален.

**В: Могу ли я отредактировать видеокружок после отправки?**
А: Нет, видеокружки отправляются как завершенные файлы. Вы можете только удалить сообщение и отправить новый видеокружок.

**В: Сколько времени загружается видеокружок?**
А: Время зависит от размера файла и скорости соединения. Видеокружок длительностью 10 секунд обычно загружается за 2-5 секунд на хорошем соединении.
